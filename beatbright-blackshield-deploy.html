Deployment Checklist

Use this exact order so the page goes live cleanly on the Blackshield Institute website.

1. Save this file as beatbright-blackshield-deploy.html or rename it to match your site routing.
2. Replace shield-logo.png with the actual path of your Blackshield shield logo asset.
3. Upload the file into the site folder where you want the BeatBright experience to live.
4. Add a menu link from your main Blackshield Institute navigation to this page.
5. Test in Chrome on Chromebook first. Click once to unlock audio, then press Play.
