# BLACKSHIELD PRODUCT STAGING AREA

Purpose:
This folder temporarily holds deployable Blackshield products created in other development environments (chats) before they are integrated into the live platform.

Rules:

1. No direct deployment from external sources into production folders.
2. All incoming assets must first be placed in this staging area.
3. Each asset must be evaluated for:
   - curriculum alignment
   - deployment readiness
   - duplication conflicts
   - routing placement

After validation, products will be moved into:

interactive/
curriculum/
passages/
downloads/
or platform modules.
