# Security Policy

## Public Static Site Notice

This repository is deployed as a static GitHub Pages website.

Anything placed in this public repository can be viewed publicly.

Do not store:

- Passwords
- Private customer data
- Learner records
- Payment records
- Admin credentials
- Private manuscripts
- Protected downloads
- Unpublished restricted product materials

## Admin Pages

Any admin panel, upload console, dashboard, or backend-style screen in this repository must be treated as a demo unless real authentication and backend security are added.

GitHub Pages does not protect admin pages by itself.

## Reporting Issues

Use the contact page or repository issue tracker for non-sensitive reports.

Do not post private information in public GitHub issues.
